# Linear-Regression-Model
📌 Project Overview

This project focuses on predicting individual medical insurance costs using Machine Learning. The goal is to build a regression model that can estimate insurance charges based on personal and health-related attributes.

🎯 Problem Statement

Insurance companies need to estimate medical costs accurately to manage risk and pricing strategies. In this project, we develop a Linear Regression model that predicts insurance charges using factors such as age, BMI, smoking habits, and number of dependents.

📊 Dataset Description

The dataset contains records of individuals along with their demographic and health-related details. Each record represents a person and their corresponding medical insurance cost.

📌 Features

age – Age of the individual
sex – Gender (male/female)
bmi – Body Mass Index (health indicator based on weight and height)
children – Number of dependents covered
smoker – Smoking status (yes/no)
region – Residential area (northeast, southeast, southwest, northwest)
charges – Medical insurance cost (target variable)

⚙️ Objective

Perform data preprocessing and analysis
Train a Linear Regression model
Evaluate model performance
Predict insurance costs based on input features

🧠 Approach

The project involves data cleaning, encoding categorical variables, splitting the dataset, training the model, and evaluating it using regression metrics.

📈 Expected Outcome

A predictive model that can estimate medical insurance charges and help understand how different factors impact healthcare costs.
