# 💰 Insurance Cost Prediction

---

## 📌 Project Overview

This project focuses on predicting **medical insurance charges** using a **Linear Regression model**.  
The model estimates insurance costs based on personal and health-related attributes such as age, BMI, smoking status, and number of dependents.

---

## 🎯 Problem Statement

Insurance companies need accurate cost estimation to manage **risk, pricing, and policy decisions**.

In this project, we build a machine learning model that:
- Predicts individual medical insurance charges  
- Identifies key factors affecting healthcare costs  

---

## 📊 Dataset Description

The dataset contains records of individuals along with their demographic and health-related information.

Each row represents:

➡️ A person  
➡️ Their attributes  
➡️ Their corresponding insurance cost  

---

## 📌 Features

| Feature | Description |
|--------|------------|
| age | Age of the individual |
| sex | Gender (male/female) |
| bmi | Body Mass Index (health indicator) |
| children | Number of dependents covered |
| smoker | Smoking status (yes/no) |
| region | Residential region (NE, SE, SW, NW) |
| charges | Medical insurance cost (target variable) |

---

## ⚙️ Project Objectives

- Perform **data preprocessing and cleaning**  
- Conduct **Exploratory Data Analysis (EDA)**  
- Train a **Linear Regression model**  
- Evaluate model performance using regression metrics  
- Predict insurance costs for new data  

---

## 🧠 Approach

The project follows a standard machine learning pipeline:

1. Data Collection  
2. Data Cleaning  
3. Encoding Categorical Variables  
4. Train-Test Split  
5. Model Training (Linear Regression)  
6. Model Evaluation  
7. Prediction  

---

## 📈 Expected Outcome

- A predictive model that estimates insurance charges  
- Understanding of how features like **age, BMI, and smoking** impact cost  
- Insights useful for insurance pricing and decision-making  

---

## 🛠️ Technologies Used

- Python  
- NumPy  
- Pandas  
- Matplotlib / Seaborn  
- Scikit-learn  

---

## 📌 Conclusion

This project demonstrates how **Linear Regression** can be applied to real-world healthcare data to predict insurance costs and extract meaningful insights.
